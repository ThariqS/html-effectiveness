# Contributing

Thanks for your interest in contributing.

This repository is a curated gallery of standalone HTML examples that accompany a
blog post. Because the examples are referenced from published writing, changes
are reviewed carefully and kept intentionally small in scope.

## How to contribute

- **Issues** — Bug reports (broken rendering, typos, accessibility problems) and
  small suggestions are welcome. Please open an issue describing what you saw and
  the browser you used.
- **Pull requests** — For anything beyond a trivial fix, please open an issue
  first so we can discuss the change before you invest time in it.

## Ground rules

- Every example must remain a **single, self-contained `.html` file** with no
  build step and no external runtime dependencies.
- Keep all sample data fictional and obviously illustrative. Do not introduce
  real customer names, real metrics, or real internal data.
- Match the existing structure and the index in `index.html` when adding or
  renaming examples.
